blendState(weight, translation, rotation)
Interpolate the current state to another state.
This function interpolates the current state (relative translation and rotation) of the bone instance to another state of a given weight.
Parameters:
{number} weight
The blending weight.
{Cal3D.CalVector} translation
The relative translation to be interpolated to.
{Cal3D.CalQuaternion} rotation
The relative rotation to be interpolated to.
