{boolean} blendCycle(id, weight, delay)
Interpolate the weight of an animation cycle.
This function interpolates the weight of an animation cycle to a new value in a given amount of time. If the specified animation cycle is not active yet, it is activated.
Parameters:
{number} id
The ID of the animation cycle to be blended.
{number} weight
The weight to interpolate the animation cycle to.
{number} delay
The time in seconds until the new weight should be reached.
Returns:
{boolean} true if succeeded; false if any error happened.
{boolean} clearCycle(id, delay)
Fade an animation cycle out.
This function fades an animation cycle out in a given amount of time.
Parameters:
{number} id
The ID of the animation cycle to be faded out.
{number} delay
The time in seconds until the the animation cycle is completely removed.
Returns:
{boolean} true if succeeded; false if any error happened.
{boolean} executeAction(id, delayIn, delayOut, weightTarget, autoLock)
Execute an animation action.
This function executes an animation action.
Parameters:
{number} id
The ID of the animation action to be execute.
{number} delayIn
The time in seconds until the animation action reaches the full weight from the beginning of its execution.
{number} delayOut
The time in seconds in which the animation action reaches zero weight at the end of its execution.
{number} weightTarget
The weight to interpolate the animation action to.
{boolean} autoLock
This prevents the action from being reset and removed on the last keyframe if true.
Returns:
{boolean} true if succeeded; false if any error happened.
