{number} calculateNormals(submesh, normalBuffer, stride)
Calculate the transformed normal data.
This function calculates and returns the transformed normal data of a specific submesh.
Parameters:
{Cal3D.CalSubmesh} submesh
The submesh from which the normal data should be calculated and returned.
{Array} normalBuffer
The user-provided buffer where the normal data is written to. This is an output parameter.
{number} stride
(Optional). The offset from the start of one data to the start of next data, default to 3.
Returns:
{number} The number of normals written to the buffer.
