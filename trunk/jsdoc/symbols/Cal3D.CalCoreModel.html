{boolean} addAnimationName(name, coreAnimationId)
Create or overwrite a string-to-animation ID mapping.
This function makes an animation ID reference-able by a string name.
Note that we don't verify that the ID is valid because the animation may be added later. Also, if there is already a helper with this name, it will be overwritten without warning.
Parameters:
{string} name
The (new) name that will be associated with the ID.
{number} coreAnimationId
The ID of the animation to be referenced by the given name.
Returns:
{boolean} true if succeeded; false if any error happened.
